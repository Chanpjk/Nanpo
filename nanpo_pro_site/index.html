Core Tweaks

Rebuild Spotlight Index

Fix slow search results by rebuilding Spotlight.

  1. Open System Settings ▸ Siri & Spotlight.
  2. Toggle Spotlight off and on.
  3. Or run: sudo mdutil -E / in Terminal.

Trim Login Items

Stop heavy apps from launching at boot.

  1. Go to System Settings ▸ General ▸ Login Items.
  2. Disable unnecessary apps, remove background items.
  3. Reboot to feel the difference.

Storage & Caches

Recover space the safe way.

  1.  ▸ About This Mac ▸ Storage ▸ Manage.
  2. Enable Empty Trash Automatically, remove large files.
  3. Clear app caches: ~/Library/Caches (quit apps first).

Reduce Visual Effects

Snappier UI on older Macs.

  1. System Settings ▸ Accessibility ▸ Display.
  2. Enable Reduce motion and Reduce transparency.

Install Homebrew

Get developer tools in one command.

  1. Open Terminal.
  2. Run: /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
  3. Then: brew install --cask iterm2 rectangle

Safari Performance

Faster browsing.

  1. Safari ▸ Settings ▸ Privacy: disable cross‑site tracking.
  2. Extensions: remove unused add‑ons.
  3. Clear History ▸ All History.

Frequently Asked

Is this safe?

All tweaks shown are native system options or simple file‑level cache clears. You can revert everything in System Settings.

Which macOS versions are supported?

Tested on Ventura, Sonoma and Sequoia. Some menu wording may vary slightly.

Do I need admin rights?

Only for a few steps (Spotlight reindex, some cache clears) and for first app launch.
